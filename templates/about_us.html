Tweet Creep

Christopher J. Lowetz, Hunter G. McElroy, Chungwei Yen

For our CSCE 470 final project, we propose building a lightweight geolocation tool that enables crowd pattern analysis in realtime.

Functionality

Tweet Creep is a lightweight, Twitter-based geolocation web service that enables the user to:

  • see who is tweeting nearby by mapping the most recent tweets
  • analyze crowd movements by viewing the last 5 tweets of each mapped user
  • view metadata for each tweet in realtime, including a current address provided by Google Maps API

    • User Needs

    People who like to Tweet and "check in" would like our web application. We can see where others have been and tweeted before on our map.

    Existing Tools

    There are a number of similar geolocation tools for Twitter, but many have not been updated since the major Twitter API changes in 2012. For example, the service "Tweography" (http://www.tweography.com/) has lost all functionality due to the change. Our tool will be up-to-date and use the current Twitter API.

    Our web application will be focused on mapping strictly local tweets and searching for individual twitter users. Many of the available solutions focus on trends and map tweets from around the world. Our tool will show users where twitter users are (and where they are headed) in their local area. People will care about this difference because most users only care about what is going on in their area.

    Building TWEET CREEP is nontrivial. The primary challenge of building the tool lies in be interfacing the Twitter data with Google Maps.

    Other similar tools can be found at:

  • http://onemilliontweetmap.com/
  • http://worldmap.harvard.edu/tweetmap/

    • Implementation

    The backend, written in Python, will make API calls to Twitter and Google Maps. Tweet Creep relies on Twitter for data, which will managed by a core algorithm based on TF-IDF weighting and Cosine Similarity.

    The frontend, written in HTML, will use Google Maps API to display data on a Google Map. Visual goodies will also come from HTML.

    Existing Resources

    Tweet Creep will make use of the Google Maps and Twitter APIs with Python support provided by Google App Engine.

    Demonstrating Usefulness

    The usefulness of Tweet Creep will be easily demonstrated in social contexts, particularly by crowd analysis on Northgate. Tweet Creep can be used to view crowd patterns, so that the user may find the most docile watering hole (or, more likely, the wildest party).
